## 📢 프로젝트 소개

PINGER(핑거)는 **카카오맵 API**를 활용한 혁신적인 구직 플랫폼입니다.
공공기관 취업을 준비하는 구직자들을 위해, 기획재정부 공공기관 채용정보 API를 활용하여 실시간 채용 리스트를 제공하며, 공고별 위치를 지도에 핑으로 표시합니다.
단순한 채용 정보 제공을 넘어, 구직자는 해당 공고에 맞춰 자기소개서를 작성하고, 멘토의 검토와 피드백을 받을 수 있어 더욱 체계적인 취업 준비가 가능합니다. 또한, 구글과 카카오
소셜 로그인을 지원해 번거로운 가입 과정 없이 간편하게 시작할 수 있습니다. <br><br>
PINGER와 함께 손가락 터치 한 번으로 채용 정보를 빠르게 확인하고, 더 빠르고 효율적인 구직 경험을 만나보세요.

## 🚀 주요 기능

📝 **회원가입 / 로그인**
- 사용자는 이메일, 비밀번호, 닉네임, 희망 근무지역, 역할(구직자/멘토)을 입력하여 회원가입이 가능합니다.
- 회원가입 시 클라이언트 상태 관리는 Zustand를 사용해 관리하며, 회원의 정보는 실시간으로 상태에 반영됩니다.
- 소셜 로그인 기능으로 구글과 카카오 로그인을 지원하며, 해당 기능은 현재 구현 중에 있습니다.

🏠 **홈 (지도 화면)**
- 로그인 시 사용자가 선택한 지역이 초기 좌표로 설정됩니다. (기본 초기 좌표: 서울)
- 공공기관 채용정보 API를 활용하여 지도에 채용 정보를 마커(Marker) 형태로 표시합니다.
- 사용자가 마커를 클릭하면 해당 채용 정보에 대한 정보창이 마커 위에 나타나며, `자세히 보기` 버튼을 클릭 시 해당 채용 정보의 상세 페이지로 이동합니다.
- 검색창에 특정 회사명을 입력 시 해당 회사의 채용 정보를 지도에서 확인할 수 있습니다.
- 검색창에 특정 지역을 입력 시 해당 지역의 채용 정보 리스트가 출력되며, 검색 결과를 클릭하면 해당 회사의 위치로 이동하며 정보창이 표시됩니다.

👷🏻‍♂️ **채용공고 리스트 / 상세 페이지**
- 공공기관 채용공고 API의 데이터를 기반으로 채용공고 리스트를 보여줍니다.
- 마음에 드는 공고를 북마크할 수 있고 해당 공고에 지원한 자소서의 개수를 볼 수 있습니다.
- 위치 지도와 함께 채용 공고에 대한 자세한 정보를 확인할 수 있으며, 기업에 대한 후기를 자유롭게 남길 수 있습니다.

✏️ **자기소개서 작성**
- 채용공고 리스트에서 본인이 지원하고 싶은 기업을 선택하고, 그 기업에 대한 자기소개서를 작성할 수 있습니다.

📜 **자기소개서 리스트 / 상세 페이지**
- **멘토**: 작성된 전체 자기소개서를 확인할 수 있습니다. 구직자들이 작성한 자기소개서를 보고 검토할 수 있습니다.
- **구직자**: 내가 작성한 자기소개서를 확인할 수 있습니다. 자기소개서 수정, 삭제가 가능합니다. (멘토가 검토한 이후 수정 불가)

👋🏻 **마이페이지** 
- 사용자가 닉네임과 주소를 수정할 수 있습니다.
- Role에 따라 다른 정보가 제공됩니다.
    - **멘토**: 자신이 검토 완료한 자소서를 확인할 수 있습니다.
    - **구직자**: 북마크한 채용 정보를 확인할 수 있으며, 북마크 취소가 가능합니다.
